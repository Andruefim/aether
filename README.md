# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/f66e3cd3-949a-429f-96c8-c1305b3d27e5

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Run [Ollama](https://ollama.com) locally and pull the model: `ollama pull qwen3:latest`
3. Run the app:
   `npm run dev`
